# PristinePlayer

厌倦了某云音乐，没用的功能太多，广告太多。于是打算写一个纯粹的，原始的音乐播放器。

# 功能

1、读取手机所有音乐，音乐所处的第一父文件夹作为本地歌单。

2、正常音乐播放，可设置播放模式。
